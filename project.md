# Проект: сайт YouTube-блогера

## Цель
Сделать одностраничный сайт YouTube-блогера с тёмной темой, где:
- показываются последние видео (автоматически через YouTube API),
- отображается информация о канале,
- есть блок «О себе» и ссылки на поддержку.

## Технологии
- HTML5 + CSS3 + Vanilla JavaScript
- YouTube Data API v3
- Развёртывание: GitHub Pages

## Структура сайта
1. **Hero (Главная)**
   - Аватар/обложка
   - Имя блогера
   - Краткий слоган
   - Кнопка перехода на YouTube-канал

2. **О себе**
   - Краткая биография
   - Описание тем и формата контента

3. **Видео**
   - Сетка карточек последних видео
   - Превью, заголовок, дата публикации
   - Ссылка на видео YouTube

4. **Статистика канала**
   - Подписчики
   - Просмотры
   - Количество видео

5. **Донат / Поддержка**
   - Кнопки на Boosty / Patreon / DonationAlerts (или другие платформы)

## Дизайн
- Тёмная тема (базовый фон #0f0f0f)
- Акценты в стиле YouTube (#ff0000)
- Адаптивная верстка (mobile-first)
- Плавные hover-эффекты

## Структура файлов

/

## Интеграция с YouTube API
Используем YouTube Data API v3:
- `channels` — получить статистику канала
- `playlistItems` — получить последние видео из uploads-плейлиста

Нужны параметры:
- `apiKey`
- `channelId`

## Безопасность
- API-ключ хранить в `config.js` только с ограничениями в Google Cloud:
  - HTTP referrer restrictions (домен сайта)
  - Ограничение только на YouTube Data API v3
- Никогда не использовать ключи с широкими правами.

## План реализации
1. Создать HTML-структуру страницы
2. Добавить тёмную адаптивную стилизацию
3. Подключить YouTube API и вывести статистику
4. Вывести карточки последних видео
5. Добавить блок донатов и соцссылок
6. Проверить и задеплоить на GitHub Pages
